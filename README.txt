Group Name: Co-Shooter

Group Members: Connor West, Harsh Kothari, Matthew Mohaupt

Brief game idea: The host player will be presented questions on a billboard and will be given the opportunity to answer by shooting the correct answer. The second player will cooperate with the first player by sharing hints that the second player finds within their area. Once the questions are answered correctly the players can steal the gold!

Player Interaction Pattern: Cooperative play

Objective: Solution
Serious Objective: Educate players about CPU architecture

Procedures: Players will be communicating with each other with hints to solve questions and shoot the right answers

Rules: Only one player can see the hints and one can shoot.

Resources: Limited by your own knowledge of the concepts and hints

Non-plain-vanilla procedure/rule: Players must communicate outside of the game (In person/Over call)

Test Question for Serious Objective: What is a CPU?

Expected Correct Answer for Serious Objective: Central Processing Unit
